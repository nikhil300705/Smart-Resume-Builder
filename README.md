# 🤖 Smart Resume Builder with AI Suggestions

A full-stack AI-powered resume builder that helps users create professional resumes, receive intelligent suggestions, and optimize content for ATS (Applicant Tracking System).

## 🚀 Features
- AI-powered resume suggestions
- ATS optimization
- Section-wise editing
- Job matching
- PDF download

## 🧠 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
